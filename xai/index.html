텍스트 · NLP 설명기법

Integrated Gradients(텍스트)

기준 임베딩부터 실제 임베딩까지 경로적분한 토큰별 기여도

반사실 · 생성 기반

반사실 텍스트 생성

Rationale Extraction

예측을 정당화하는 최소 부분 문장(rationale) 추출

정형데이터 특화 설명

표 형태 데이터는 특징 간 상관관계가 강해서 순진한 특징 중요도 계산이 쉽게 왜곡됩니다.

의존도 시각화

PDP(부분의존도 플롯)

특징 하나를 바꿔가며 평균 예측 변화를 그린 전역 곡선

ICE plot

PDP를 개별 샘플 단위로 쪼갠 곡선, 샘플 간 이질적 효과 포착

ALE

상관관계 있는 특징에서 PDP보다 안전, 국소 구간 차분을 누적

규칙 · 트리 특화

Anchors

if-then 규칙, 조건 충족 시 예측이 거의 바뀌지 않음을 보장하는 고신뢰 설명

TreeSHAP

트리 구조를 활용해 다항시간에 정확한 SHAP값을 계산하는 고속 변형

공정성 · 편향 설명

설명가능성은 모델이 어떤 특징에 의존하는지 보여주는 것을 넘어, 그 의존이 특정 집단에 불공정한 영향을 주는지 드러내는 데도 쓰입니다.

격차 진단

보호집단 간 예측률 비율 비교

Equalized Odds 점검

집단별 참양성률 · 위양성률 차이 측정

반사실 · 소스 추적

반사실 공정성

보호속성만 바꾼 반사실 개체의 예측이 동일한지 검증

SHAP 기반 편향 소스 추적

민감속성과 상관된 대리 특징의 기여도를 분리해 편향 경로 파악

실무 도구 · 라이브러리

논문 속 기법을 실제 코드로 옮길 때 쓰는 오픈소스 라이브러리들입니다.
