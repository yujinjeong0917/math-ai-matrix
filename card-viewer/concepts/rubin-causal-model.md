## 기본설명
개체 $i=1,\dots,n$에 대해 이진 처치변수 $T_i\in\{0,1\}$과 두 잠재결과 $Y_i(1), Y_i(0)$을 정의하자. 관측된 결과는 $Y_i = T_iY_i(1)+(1-T_i)Y_i(0)$ 로 주어진다. 이때 개체수준 인과효과 $\tau_i = Y_i(1)-Y_i(0)$ 는 어떤 개체 $i$에 대해서도 $Y_i(1)$과 $Y_i(0)$ 둘 다를 동시에 관측하는 것이 불가능하므로 데이터만으로는 식별되지 않는다(근본문제, fundamental problem of causal inference).

## 문제
실제로 관측되는 결과 $Y_i$는 처치지시 $T_i$에 따라 두 잠재결과 중 하나를 '선택'하는 함수여야 한다. 이를 지시변수를 이용해 하나의 식으로 쓰면 $Y_i = $==빈칸== 이다.

## 해설
$T_i=1$이면 첫째 항만 남아 $Y_i(1)$이 되고, $T_i=0$이면 둘째 항만 남아 $Y_i(0)$이 되도록, 두 경우를 지시변수 곱으로 이어붙인 것입니다.

**정답: $T_iY_i(1) + (1-T_i)Y_i(0)$**

## 예시
신약 임상시험에서 환자 두 명을 생각해보자. 환자 A는 실제로 약을 받았고($T_A=1$), 만약 약을 받았다면 혈압이 $Y_A(1)=118$mmHg, 받지 않았다면 $Y_A(0)=132$mmHg이었을 것이라 하자(둘 다 개념상의 참값). 환자 B는 약을 받지 않았고($T_B=0$), $Y_B(1)=125$, $Y_B(0)=136$이라 하자.

관측된 데이터는 스위칭 방정식으로 결정된다.

$$Y_A = T_A Y_A(1) + (1-T_A)Y_A(0) = 1\times118 + 0\times132 = 118$$

$$Y_B = T_B Y_B(1) + (1-T_B)Y_B(0) = 0\times125 + 1\times136 = 136$$

즉 실제 데이터셋에는 $(T_A=1,\ Y_A=118)$과 $(T_B=0,\ Y_B=136)$만 기록되고, $Y_A(0)=132$나 $Y_B(1)=125$는 영원히 관측되지 않는다. 개체수준 효과 $\tau_A=118-132=-14$, $\tau_B=125-136=-11$이 참값이라 해도, 데이터만으로는 이를 확인할 방법이 없다.
