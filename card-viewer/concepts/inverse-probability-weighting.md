## 기본설명
무교란성 $(Y_0,Y_1)\perp T\mid X$ 와 양성성 $0&lt;e(X)&lt;1$ 하에서, 참 성향점수 $e(X)=P(T=1\mid X)$를 그대로 안다고 할 때 $\psi(W) := \dfrac{TY}{e(X)}-\dfrac{(1-T)Y}{1-e(X)}$ 는 $E[\psi(W)] = \mathrm{ATE} := E[Y_1-Y_0]$ 을 만족하는 불편추정함수다.

## 문제
$T=1$인 사람만 실제로 $Y=Y_1$이 관측되므로, $TY$라는 곱은 $T$가 $0$일 때 자동으로 $0$이 되어 $T=1$일 때의 값만 남긴다. 따라서 $TY = TY_1$ 이라 쓸 수 있고, 이를 조건부기댓값에 그대로 대입하면 $E[TY\mid X] = $==빈칸== 이다.

## 해설
$T=1$일 때 관측되는 $Y$는 정의상 $Y_1$이고, $T=0$일 때는 $TY=0\times Y=0$이라 어떤 값이든 상관없어요. 그러니 $TY$와 $TY_1$은 항상 같은 확률변수이고, 조건부기댓값도 그대로 같아요.

**정답: $E[TY_1\mid X]$**

## 예시
추상적 증명 전에 이산 공변량으로 각 층에서 조건부기댓값이 실제로 정확히 맞아떨어지는지 확인해봅니다.

$X\in\{0,1\}$이 각각 확률 $0.5$로 나오고, 참 성향점수는 $e(0)=0.4,\ e(1)=0.6$, 참 결과함수는 $\mu_1(0)=5,\ \mu_1(1)=8,\ \mu_0(0)=3,\ \mu_0(1)=4$ 라 합시다. 그러면 참 ATE는 다음과 같습니다.
$$\mathrm{ATE} = E[\mu_1(X)-\mu_0(X)] = 0.5\times(5-3)+0.5\times(8-4) = 1+2 = 3$$
이제 처치 항 $TY/e(X)$의 조건부기댓값을 각 층에서 직접 계산해봅니다. $X=0$층에서는 $E[TY/e(X)\mid X=0]=\mu_1(0)=5$이고 $X=1$층에서는 $\mu_1(1)=8$입니다 (증명 s2~s3에서 이 등식이 왜 성립하는지 보입니다). 이 둘을 $X$의 분포로 평균내면 $0.5\times5+0.5\times8=6.5=E[Y_1]$이 됩니다. 대칭적으로 대조 항 $(1-T)Y/(1-e(X))$은 같은 방식으로 $0.5\times3+0.5\times4=3.5=E[Y_0]$이 됩니다.

둘을 빼면 $6.5-3.5=3$으로 참 ATE와 정확히 일치합니다. 아래 증명은 이 일치가 이 숫자 예제만의 우연이 아니라 항상 성립하는 대수적 사실임을 보입니다.
