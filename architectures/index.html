YOLO부터 Transformer까지
모델 아키텍처 심화

특정 모델을 택소노미로 분류하고, 순전파 시퀀스를 그대로 따라가며, 내부 연산을 전부 수식으로 증명하고, 버전이 바뀌며 무엇이 달라졌는지까지 보는 걸 목표로 합니다. 2D 객체탐지, Transformer, 3D, 시계열을 우선순위로 잡았습니다.

참고. 지금은 이정표와 핵심 연산만 정리된 설계 초안입니다. 각 모델의 순전파 시퀀스 다이어그램과 빈칸 증명 페이지는 다음 단계에서 이어갑니다.
객체탐지 (2D) Transformer 계열 3D · 포인트클라우드 시계열 CNN 백본 계보 Diffusion 아키텍처 State-Space · 효율적 시퀀스 멀티모달 아키텍처

대분류 안에 중분류, 그 안에 구체 기법 카드가 있는 구조입니다. 아직 빈칸 증명 페이지는 없고 개념 정리 단계입니다.

객체탐지 (2D) · YOLO 계열

버전마다 무엇이 바뀌었는지를 수식 단위로 추적하는 게 목표입니다. 지금은 이정표만 잡아뒀습니다.

CNN 백본 계보

합성곱 신경망이 얕은 구조에서 매우 깊고 효율적인 구조로 진화해온 과정을 정리합니다.

백본 진화 이정표

LeNet/AlexNet

초기 CNN 구조, 합성곱과 풀링을 쌓아 이미지 특징 추출

ResNet

residual connection으로 매우 깊은 네트워크 학습 가능하게 함

EfficientNet

깊이 · 너비 · 해상도를 동시에 스케일링하는 compound scaling

Diffusion 아키텍처

노이즈를 점진적으로 제거하며 이미지를 생성하는 모델의 백본 구조를 다룹니다.

핵심 구성요소

U-Net

인코더-디코더 + skip connection, 노이즈 예측 네트워크의 표준 백본

Latent Diffusion

픽셀이 아닌 압축된 잠재공간에서 확산을 수행

텍스트 조건화

cross-attention으로 텍스트 임베딩을 U-Net에 주입

State-Space · 효율적 시퀀스 모델

Transformer의 이차 연산량 문제를 상태공간모델로 우회하는 계열입니다.

핵심 아키텍처

Mamba

선택적 상태공간모델, 시퀀스 길이에 선형적인 연산량

S4

구조화된 상태공간, HiPPO 초기화로 장기의존성 포착

추론 특성

Transformer 대비 KV 캐시 없이 상수 메모리로 추론 가능

멀티모달 아키텍처

이미지와 텍스트처럼 서로 다른 모달리티를 하나의 표현공간에서 다루는 구조입니다.

정렬 · 융합 방식

CLIP

이미지-텍스트를 같은 임베딩 공간에 대조학습으로 정렬

VLM 이미지 토큰화

이미지를 패치 임베딩으로 잘라 언어모델 입력 토큰처럼 사용

크로스모달 어텐션

서로 다른 모달리티 표현을 어텐션으로 융합
